# Student Performance Prediction System + UI (React + FastAPI)

## Overview

The **Student Performance Prediction System** is a machine learning project developed using **Python** and **Scikit-learn** to predict a student's **Exam Score** based on various academic and behavioral factors. The model analyzes historical student data to identify patterns and estimate future exam performance.

This project demonstrates the complete machine learning workflow, including **data preprocessing**, **exploratory data analysis (EDA)**, **model training**, **evaluation**, **visualization**, and **prediction**.

---

## Features

- Predicts students' exam scores using Machine Learning.
- Performs data preprocessing and feature selection.
- Includes Exploratory Data Analysis (EDA) with visualizations.
- Trains a **Random Forest Regression** model.
- Evaluates model performance using regression metrics.
- Saves the trained model for future predictions.
- Supports prediction for new student data.

### Input Features
